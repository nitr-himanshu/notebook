# Markdown

## Heading

| Syntax | Description |
| ---------- | ----------|
|\# H1 | Heading 1 |
|\#\# H2 | Heading 2 |
|\#\#\# H3 | Heading 2 |

**e.g.**

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

## Bold

\*\*bold text\*\*

**I am bold.**

## Italic

\*italic\*

*I am italic*

## Blockquote

\> block quote

> I am block quote

## Ordered List

1\. Item

...

**e.g.**

1. item 1
2. item 2
3. item 3
4. item 4

## Unordered List

\- Item

**e.g.**

- item 1
- item 2
- item 3

## Code

\`code\`

**e.g.**

`My code`

## Horizontal Rule

\---

**e.g.**

---

## Link

\[title](<https://www.example.com>)

**e.g.**

[Google](https://www.google.com)

## Image

\!\[alt text](image.jpg)

**e.g.**

![github](img/github.png)

### Resize image

<img src="img/github.png" alt="github_resized" style="width:200px;"/>

### GIF images

<img src="img/minion.gif" alt="minion" style="width:200px;"/>
